What's there to say?

Right now I'm diving into AWS, so what better way to pick things up than to play around with a web site?

I know this is a simple HTML site, but behind the scenes, I upload a text file for every page I want to create and based on the folder its placed in, it will shove that copy into a template and copy to my production bucket.

Conversely, if I want to update the templates, I can drop my new template in a folder and it will automatically regenerate all of my existing pages so that way they have the new looking and feel.
